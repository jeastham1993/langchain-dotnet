# Chroma

Chroma is an open-source Apache 2.0 embedding database.

## Setup

1. Run chroma with Docker on your computer [docs](https://docs.trychroma.com/api-reference)
2. Install the Chroma JS SDK.

```bash npm2yarn
npm install -S chromadb
```
